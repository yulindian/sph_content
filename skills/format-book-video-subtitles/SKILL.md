# 短视频字幕排版

## 输入

接收书名和完整口播文案。书名用于识别文案中的完整标题，不能凭空添加到没有书名的文案中。

## 目标

将输入的长文案拆分为纯文本字幕。每行字数以7至9字为最佳，但不要为了凑字数固定切成9字；必须优先服从句意、口语朗读节奏和配音气口。最终文本必须剔除所有标点符号，并保持语义完整。

## 核心规则

- 删除全部标点符号，但不删除原文中的汉字、数字、英文字母及有意义内容。
- 每行以7至9个字符为最佳，普通行绝不超过9个字符。
- 在自然语义停顿或配音气口处切分。
- 一个意思讲完就换行；意思没讲完时不要只因为达到9字就生硬换行。
- 不在完整词语、人名、地名、数字、年代、固定概念中间切分。
- 助词“的”不得位于行首。
- 每个字幕短句单独一行，保持一句一行的导入结构，不插入序号、项目符号、解释或空白装饰。
- 保持原文内容顺序和叙事逻辑，不改写、不概括。

## 书名特例

- 完整书名优先保持在同一行；书名行只放书名，不混入前后句。
- 书名超过9个字符时允许该行超过9字。
- 书名在正文中每次完整出现时均按同一规则处理。
- 不把书名号保留到输出中。

## 工作流

1. **节奏拆分**：先在不改变原意的前提下，根据句意、朗读气口和短视频阅读节奏将文案打散为多个短句。
2. **排版与清洗**：删除所有标点符号，保留汉字、数字、英文和必要语义内容；普通行控制在7至9字最佳，超过9字时在自然停顿处继续拆分。
3. **书名复核**：检查所有完整书名是否独占一行；书名超过9字时保留完整书名，不强拆。
4. **终检输出**：逐行核对字数、语义完整性、是否误断词语、是否以“的”开头、是否残留标点，确认后输出。

## 终检

静默逐行检查：

- 除书名行外是否全部不超过9个字符。
- 是否优先达到7至9字，但没有固定9字切片。
- 是否有标点残留。
- 是否出现以“的”开头的行。
- 是否切断完整词语或遗漏、重复原文内容。

## 输出

仅使用一个代码块输出纯文本字幕，一句一行。不要在代码块内外添加统计、解释或其他文字。
