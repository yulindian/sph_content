# 即梦写实场景生图提示词

## 输入与对应关系

接收一段或多段带序号文案。每个输入段落只生成一个提示词：

- 数量必须完全一致。
- 序号必须一一对应。
- 不得合并、拆分或遗漏段落。
- 提示词呈现文案的核心事件和情绪，不得新增会改变原意的事实。

## 画面要素

每条提示词都用连贯自然的中文描述，并包含：

1. **时代背景**：根据文案证据判断朝代、年代或现代时期；线索不足时使用可靠的宽泛时期，不虚构精确年代。
2. **视觉锚点**：选取2至4个与时代、地点和事件相符的建筑、器物、服饰、人物动作或环境元素。
3. **情绪氛围**：例如沧桑、繁华、静谧、肃穆、市井烟火气或工业锈蚀感。
4. **色调与光源**：明确主色调以及晨光、暮色、月光、阴天柔光、博物馆顶光或侧逆光等。
5. **景别与视角**：选择远景、中景、近景或特写，并采用适合4:3横屏的水平叙事。

## 构图映射

- 远景：使用广角横向展开、平行视野延伸或开阔全景俯瞰。
- 中景：使用建筑横向排列、人物与环境水平互动或街巷水平透视。
- 近景/特写：使用黄金比例、左右三分线或横向纹理细节。
- 自然场景：匹配合理的自然光时段。
- 器物文物：使用柔和顶光、暗背景聚光或侧逆光突出材质。
- 人物：使环境光与情绪一致，避免畸形肢体、时代错置和无关人物抢占主体。

## 固定约束

每条提示词必须：

- 明确写出“4:3横屏构图”。
- 包含“8k超清电影级写实纪录片质感”。
- 结尾包含“无后期文字”。
- 禁止画面文字、字幕、Logo、水印、标注和无意义符号。
- 保持电影级写实纪录片风格，不强制油画、厚涂或其他未指定艺术风格。

## 输出

只在一个代码块中输出全部提示词。固定格式：

```text
【1】完整自然语言场景描述，4:3横屏构图，8k超清电影级写实纪录片质感，无后期文字。
```

输出前静默核对数量、序号、时代、元素、氛围、色调、视角、光影和固定后缀。
